Design System

Typography Tokens

All default type tokens for a dashboard web app — set in Source Sans Pro. Every token maps to a tailwind.config key so changing one value cascades everywhere automatically.

01 — Font Size Scale

text-2xs Dashboard overline · micro badge — 2XS 10px
text-xs Chart axis tick · table footer — XS 11px
text-sm Caption · helper text · tag labels — SM 12px
text-md Default body text · table cells · form inputs — MD 14px
text-lg Emphasized body · subheadings · nav items — LG 16px
text-xl Card titles · section leads · modal headings — XL 18px
text-2xl Widget headlines · sidebar titles — 2XL 22px
text-3xl Page titles · drawer headings — 3XL 28px

02 — Font Weight Scale

font-extralight Extralight — Decorative large text only 200

font-semibold Semibold — Labels · headings · buttons 600
font-bold Bold — Page titles · KPI numbers 700
font-black Black — Display · hero splash 900

03 — Line Height

leading-tight

Headlines & numeric displays benefit from tighter leading so stacked lines read as a single visual unit.

1.15
leading-snug

Subheadings and card titles sit at a snug rhythm — compact yet breathing slightly more than display headings.

1.30
leading-normal

Standard body text for UI components. Most dashboard labels, table cells, and form inputs default here for comfortable reading at small sizes.

1.50
leading-relaxed

Long-form body copy and description panels use a relaxed leading to improve readability across multiple sentences without fatiguing the eye.

1.65
leading-loose

Lists and spaced layouts.
Each item has generous breathing room.

2.00

04 — Letter Spacing

tracking-wide Button labels and UI elements 0.04em
tracking-wider Table headers, small labels 0.08em
tracking-widest Overlines & badge text 0.16em
